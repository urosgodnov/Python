from sklearn import datasets
from itertools import combinations as c
from scipy.stats import pearsonr
iris=datasets.load_iris()
df=pd.DataFrame(iris.data, columns=iris.feature_names)
for i in c(df.columns.tolist(),2):
    corr=pearsonr(df[i[0]],df[i[1]])[0].round(3)
    sig=pearsonr(df[i[0]],df[i[1]])[1].round(6)
    print(i[0],"and ",i[1]," corr:",corr," sig:",sig)

T-test

  • t test: compares two averages (means) and tells you if they are different from each other
  • three main types of t-test:
    • an Independent Samples t-test compares the means for two groups (stats.ttest_ind)
    • a Paired sample t-test compares means from the same group at different times (stats.ttest_rel)
    • a One sample t-test tests the mean of a single group against a known mean (stats.ttest_1samp)

Example

  • is the blood pressure from female doctors on average higher than 120
from scipy import stats
female_doctor_bps = [128, 127, 118, 115, 144, 142, 133, 140, 132, 131, 
                     111, 132, 149, 122, 139, 119, 136, 129, 126, 128]
print(stats.ttest_1samp(female_doctor_bps, 120))
## Ttest_1sampResult(statistic=4.512403659336718, pvalue=0.00023838063630967753)

Example

from scipy import stats
female_doctor_bps = [128, 127, 118, 115, 144, 142, 133, 140, 132, 131,

Linear regression (1)

  • preparing data. Independent variables have to be array. USE reshape!
  • lm.fit(): fits a linear model
  • lm.predict(): predict Y using the linear model with estimated coefficients
  • lm.score(): returns the coefficient of determination (R^2)
  • sklearn is very “poor/user unfriendly” with summary statistics and it is oriented towards machine learning

Example

from sklearn.linear_model import LinearRegression
df=df.dropna()
