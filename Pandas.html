Basic exploratory commands

  • df.head(n): n first rows
  • df.tail(n): n last rows
  • df.shape: the number of rows and columns
  • df.info(): the index, datatype and memory information
  • s.value_counts(dropna=False): would allow you to view unique values and counts for a series
  • df.describe(): summary statistics for numerical columns
  • df.corr(): correlation between columns in a data frame

Subsetting

  • df[col]: return column with label col as Series
  • df[[col1, col2]]: returns columns as a new DataFram
  • df.iloc[0,:]: selecting first row
  • df.iloc[0,0]: selecting the first element of the first column

Filter, sort and groupby

  • df[df[column] > value]: all rows where column is greater than value
  • &: and ; |: or; df[(df[column1] > value1) & (df[column2<value2])]
  • df.sort_values(column1): sorting DF by column1 ASC (default)
  • df.sort_values(column2,ascending=False): sorting DF by column1 DESC
  • df.sort_values([column1,column2],ascending=[True,False]): sorting by 2 columns
  • df.groupby(column): involves splitting the data into groups based on some criteria, applying a function to each group independently and combining the results into a data structure
  • df.groupby([column1,column2])

Data Cleaning (1)

  • pd.isnull(): checking for null values
  • df[df.column.isnull()]: returns DF with rows, where column has null values
  • pd.isnull().sum(): how many null values
  • df.dropna(): drop missing values (rows) (axis=1 columns)
  • df.fillna(x): replace missing values with x
  • s.fillna(s.mean()): replace missing values with the mean
  • df.column.fillna(df.column.mean(), inplace=True): replacing null values in column with mean of this column and saving the result immediately (inplace=True)

Data Cleaning (3)

  • s.replace(1,‘one’): replacing 1 with one

Calculated columns

  • df[calculated_column]=df[column1]+df[column2]: new column is the sum of two existing columns
  • df[calculated_column]=df.column1*df.column2: new column is the product of two existing columns
  • df[calculated_column]=pow(df.column1, 2): new column is the power of two of the column1
