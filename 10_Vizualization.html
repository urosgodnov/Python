Modules

  • Matplotlib
  • Seaborn: targets statistical data visualization
  • Seaborn is complimentary to Matplotlib
  • Seaborn:
    • creates aesthetically pleasing plots by default
    • creates statistically meaningful plots
    • understands the pandas DataFrame
    • is not directly included in Anacondas release

Seaborn plots

  • broadly divided into two classes:
    • “Axes-level” functions: regplot, boxplot, kdeplot
    • “Figure-level” functions: lmplot, factorplot, jointplot
  • axes-level functions will only draw onto an axes and wont otherwise mess with the figure
  • figure level: the functions need to have total control over the figure
  • figure level: fig and axes arguments, customization has to come after youve called the function

Lab

Import CarsCSV.csv and show Quantity by Manufacturer and Year (facet). Show only data for Volkswagen, Toyota and Skoda for year 2007 and 2008. Mind the whitespace after the value in columns. Use str.strip() to trim.

Solution

import seaborn as sns
import matplotlib.pyplot as plt
import pandas as pd

df=pd.read_csv("./Lab/Datasets/CarsCSV.csv")

df=df[["Year","Manufacturer","Quantity"]]

df=df[(df["Manufacturer"].str.strip()=="Toyota") | 
(df["Manufacturer"].str.strip()=="Volkswagen") | 
(df["Manufacturer"].str.strip()=="Skoda")]

df=df[(df["Year"]==2007) | (df["Year"]==2008)]

df=df.groupby(["Manufacturer","Year"], as_index=False).sum()

sns.catplot(x="Manufacturer", y="Quantity",  kind="bar", col="Year", data=df, height=3)
## <seaborn.axisgrid.FacetGrid object at 0x000000002FC0B4E0>

Lab

Take Titanic dataset and show survival rate according to alone variable. What is the average survival rate by alone? Is there a statistical significant difference in survival rate?

Solution (1)

import seaborn as sns
import matplotlib.pyplot as plt
import pandas as pd
from scipy import stats

titanic = sns.load_dataset("titanic")
df=titanic[["survived","alone"]]


dfa=df["survived"][df["alone"]==True]
dfna=df["survived"][df["alone"]==False]

print("Alone:",dfa.mean())
## Alone: 0.30353817504655495
print("Not alone:",dfna.mean())
## Not alone: 0.5056497175141242

Solution (2)

g = sns.catplot(x="alone", y="survived", data=df,
                height=3, kind="point")

print("T-test:",stats.ttest_ind(dfa, dfna))
## T-test: Ttest_indResult(statistic=-6.193032488079412, pvalue=9.009490179329723e-10)
