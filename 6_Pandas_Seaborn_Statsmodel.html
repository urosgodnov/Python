Filter, sort and groupby

  • df[df[column] > value]: all rows where column is greater than value
  • &: and ; |: or; df[(df[column1] > value1) & (df[column2<value2])]
  • df.sort_values(column1): sorting DF by column1 ASC (default)
  • df.sort_values(column2,ascending=False): sorting DF by column1 DESC
  • df.sort_values([column1,column2],ascending=[True,False]): sorting by 2 columns
  • df.groupby(column): involves splitting the data into groups based on some criteria, applying a function to each group independently and combining the results into a data structure
  • df.groupby([column1,column2])

Lab

  • Calculate players mean height and mean weight by birthState for players, who were born in USA after 1950.
  • Print only birthState, mean height and weight sorted by birthState DESC.
  • HINT: df.mean() returns mean for all columns.

Solution

  • selecting rows: df1=df1[(df[“birthCountry”]==“USA”) & (df[“birthYear”]>1950)]

  • renaming: df1.rename(columns={“height”:“AVheight”,“weight”:“AVweight”})

Data Cleaning

  • pd.isnull(): checking for null values
  • df[df.column.isnull()]: returns DF with rows, where column has null values
  • pd.isnull().sum(): how many null values
  • df.dropna(): drop missing values (rows) (axis=1 columns)
  • df.fillna(x): replace missing values with x
  • s.fillna(s.mean()): replace missing values with the mean
  • df.column.fillna(df.column.mean(), inplace=True): replacing null values in column with mean of this column and saving the result immediately (inplace=True)

Calculated columns

  • df[calculated_column]=df[column1]+df[column2]: new column is the sum of two existing columns
  • df[calculated_column]=df.column1*df.column2: new column is the product of two existing columns
  • df[calculated_column]=pow(df.column1, 2): new column is the power of two of the column1

Merge dataframes

  • database-style DataFrame or named Series joining/merging
  • pd.merge
  • default: inner join
  • result = pd.merge(left, right, on=‘key’, how=“inner”)
  • joining on many columns: pd.merge(left, right, on=[column1, column2])
  • joining on many columns with different names: pd.merge(left, right, left_on==[column1, column2], right_on= [column3, column4])

Example

import numpy as np
import pandas as pd

df1 = pd.DataFrame({'ID': [1,2,3,4,5,6], 'pID' : [21,22,23,24,25,26], 
'Values1' : [435,33,45,34,57,12]})

df2 = pd.DataFrame({'ID' : [4,4,5], 'pid' : [24,25,25], 
'Values2' : [544, 545, 676]})

result=pd.merge(df1, df2, left_on=['ID', 'pID'], right_on=['ID', 'pid'])

print(result)
##    ID  pID  Values1  pid  Values2
## 0   4   24       34   24      544
## 1   5   25       57   25      676

Seaborn

Modules

  • Matplotlib
  • Seaborn: targets statistical data visualization
  • Seaborn is complimentary to Matplotlib
  • Seaborn:
    • creates aesthetically pleasing plots by default
    • creates statistically meaningful plots
    • understands the pandas DataFrame
    • is not directly included in Anacondas release

Seaborn plots

  • broadly divided into two classes:
    • “Axes-level” functions: regplot, boxplot, kdeplot
    • “Figure-level” functions: lmplot, factorplot, jointplot
  • axes-level functions will only draw onto an axes and wont otherwise mess with the figure
  • figure level: the functions need to have total control over the figure
  • figure level: fig and axes arguments, customization has to come after youve called the function

Statistically meaningful plots

Linear regression

  • lmplot
  • regplot
  • parameters:
    • hue: color by categories
    • col/row: facets
    • fit_reg: regression line
    • ci: confidential intervals
    • order: polyinominal fitting curve

Import CarsCSV.csv and show Quantity by Manufacturer and Year (facet). Show only data for Volkswagen, Toyota and Skoda for year 2007 and 2008. Mind the whitespace after the value in columns. Use str.strip() to trim.

Solution

import seaborn as sns
import matplotlib.pyplot as plt
import pandas as pd

df=pd.read_csv("./Lab/Datasets/CarsCSV.csv")

df=df[["Year","Manufacturer","Quantity"]]

df=df[(df["Manufacturer"].str.strip()=="Toyota") | 
(df["Manufacturer"].str.strip()=="Volkswagen") | 
(df["Manufacturer"].str.strip()=="Skoda")]
